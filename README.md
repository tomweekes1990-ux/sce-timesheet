# SCE Timesheet Platform

A cloud-connected timesheet management platform for Specialised Civil & Electrical Pty Ltd.

## Features
- 🔐 Firebase Authentication (login/register)
- 👑 Admin panel with user management
- ☁️ Real-time cloud sync via Firestore
- 📅 Day-by-day timesheet entry
- 📄 PDF export
- 📋 Timesheet history

## Live App
Visit the deployed site at: `https://YOUR-USERNAME.github.io/sce-timesheet`

## Setup
No build step required. This is a single HTML file with all dependencies loaded via CDN.

Firebase project: `sce-timesheet-1841d`

## First Use
1. Open the app
2. Click **Register** — the first account created becomes Admin
3. Share the URL with employees to self-register

## Tech Stack
- HTML / CSS / Vanilla JS
- Firebase Auth + Firestore
- jsPDF + jsPDF-AutoTable
- Google Fonts (DM Mono, Bebas Neue)
